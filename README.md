# Social Post Project with React SSR


## Available Scripts

In the project directory, you can run:

### `yarn install` for install all dependency

### `docker-compose up -d` for create db & redis

### `yarn start`


## Tech Stack
- Redux
- React
- Styled Components
- Redux-Saga
- React-hook-form
- yup
- Jest
- Express
- Mongodb
- Redis
- Typescript
